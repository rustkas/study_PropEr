ProEr integration with types
=====

Create new OTP lib
-----
    $ rebar3 new lib ex04_types && cd ex04_types

Add information to rebar.config

Get dependencies
-----
    $ rebar3 get-deps


Get PropEr help
-----
    $ rebar3 help proper


Create new propety via PropEr
-----
    $ rebar3 new proper delete_01
