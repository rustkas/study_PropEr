An OTP library

Create new OTP lib
-----
    $ rebar3 new lib ex01_sort && cd ex01_sort

Add information to rebar.config


Get dependencies
-----
    $ rebar3 get-deps


Get PropEr help
-----
    $ rebar3 help proper


Create new propety via PropEr
-----
    $ rebar3 new proper sort
    

Format
-----
    $ rebar3 format

EUnit
-----
    $ rebar3 eunit -v

PropEr test
-----
    $ rebar3 proper
